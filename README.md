# Motivation-project-5

## Overview
This project is a simple motivational poster-style web page created using HTML and CSS.
It displays an image with an inspirational quote, styled with custom fonts and centered layout.
The goal of this project is to practice typography, layout, and basic design principles in web development.

![image alt](https://github.com/Ferb-10/Motivation-project-5/blob/61fff90552e76a3b513b803ca6c56287d81ebb80/assets/motivation-project.png)

## Features
- Motivational poster-style layout
- Centered image with a white border
- Inspirational quote displayed on top of the image
- Uses Google Fonts for custom typography
- Clean and minimal design with a dark background

## Technologies Used
- HTML5
- CSS
- Google Fonts

## What I Learned
- How to import and use Google Fonts
- Applying custom fonts using CSS classes
- Creating a centered layout using margins
- Styling images with borders and responsive sizing
- Using background colors to improve visual contrast

## 概要
このプロジェクトは、HTMLとCSSを使用して作成したシンプルなモチベーションポスター風のWebページです。
画像とインスピレーションを与えるメッセージを組み合わせ、フォントやレイアウトにこだわってデザインしました。
Webデザインにおけるタイポグラフィやレイアウトの基礎を学ぶことを目的としています。

## 機能
- ポスター風のシンプルなレイアウト
- 白枠付きの中央配置画像
- モチベーションメッセージの表示
- Google Fonts を使用したカスタムフォント
- 黒背景によるミニマルなデザイン

## 使用技術
- HTML5
- CSS
- Google Fonts

## 学んだこと
- Google Fonts の読み込みと使用方法
- CSSクラスを使ったフォント指定
- margin を使った中央レイアウトの作成
- 画像のレスポンシブ表示と枠線の追加
- 背景色によるコントラストの調整
